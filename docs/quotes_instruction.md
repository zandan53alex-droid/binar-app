# Инструкция: Система котировок (Quotes System)

Эта система обеспечивает доставку цен в реальном времени из вашего браузера в Мини-приложение в Telegram.

## 🏗 Как это устроено (Hybrid Bridge)

Система состоит из двух частей, которые "общаются" друг с другом:
1.  **Bridge (Ваш Домашний ПК)**: Скрипт `pocket_playwright.py`. Он открывает невидимое окно браузера, заходит на платформу, берет оттуда котировки и пересылает их на ваш сервер.
2.  **Hub (Ваш Сервер VPS)**: Скрипт `quotes_server.py`. Он работает как распределительный центр — принимает данные от ПК и раздает их всем пользователям, которые открыли приложение в Telegram.

---

## 🛠 Настройка и запуск (Шаг за шагом)

### 1. Настройка на вашем ПК (Откуда идут цены)
Чтобы у ваших пользователей всегда были свежие цены, ваш компьютер должен быть включен и на нем должен работать "мост":
1.  Установите **Python** на свой компьютер.
2.  Установите нужные библиотеки (в терминале/командной строке):
    ```bash
    pip install playwright websockets
    playwright install chromium
    ```
3.  Откройте файл `pocket_playwright.py` в Блокноте и в самом верху (в разделе `CONFIG`) укажите IP вашего сервера:
    `VPS_IP = "ВАШ_IP_СЕРВЕРА"`
4.  Запустите скрипт: `python pocket_playwright.py`.
5.  Когда увидите лог `💓 ПУЛЬС: ... | VPS: OK`, значит цены успешно уходят на сервер.

### 2. Настройка на сервере (Куда приходят цены)
На сервере уже должен работать сервис `quotes_service`.
- **Файл на сервере**: `/root/bot/quotes_server.py`
- **Nginx**: Нужно убедиться, что запросы через путь `/ws` перенаправляются на внутренний порт порта `8765`. Это прописано в `ultimate_setup_guide.md`.

## 🚦 Статус-баджи в приложении
Пользователи видят индикатор в углу экрана:
- **Зеленый (Connected)**: Всё отлично, цены приходят в реальном времени.
- **Желтый (Connecting)**: Приложение пытается достучаться до сервера.
- **Красный (Offline)**: Цены не поступают. Обычно это значит, что у вас выключен скрипт на домашнем ПК.

## 🆘 Если что-то не работает

| Что видим | Что проверить |
| :--- | :--- |
| **Цены стоят на месте** | Работает ли `pocket_playwright.py` на вашем ПК? Нет ли там ошибок в терминале? |
| **В приложении написано Offline** | Запущен ли сервис на VPS? Введите: `systemctl status quotes_service`. |
| **Ошибка Connection Closed** | Возможно, на платформе обновилась верстка или сессия. Просто перезапустите скрипт на ПК. |
